# 半導体集積デバイス工学研究室 Website

静的な研究室ホームページです。HTML / CSS / JavaScript だけで構成し、GitHub Pages で公開できます。

## ローカル確認

`index.html` をブラウザで開くか、任意の静的サーバーで確認してください。

## GitHub Pages への公開

1. このフォルダを GitHub の新規リポジトリへ push します。
2. GitHub の **Settings > Pages** で、Source に **GitHub Actions** を選択します。
3. 数分後、表示された URL でサイトが公開されます。

公開前に、連絡先・詳細な住所・業績を実データに置き換えてください。
